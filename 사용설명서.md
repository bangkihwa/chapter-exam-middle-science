# 목동에이원과학학원 프리미엄내신관리 시스템
## 사용 설명서

---

## 🎯 시스템 소개

**목동에이원과학학원 프리미엄내신관리 시스템**은 학생 개개인의 학습 효율을 극대화하기 위해 개발된 차세대 학습 분석 플랫폼입니다.

단순한 채점을 넘어, **16개 학교의 실제 기출문제 579문항**을 기반으로 학생의 강점과 약점을 정밀하게 분석하여 맞춤형 학습 가이드를 제공합니다.

---

## ✨ 시스템의 6가지 핵심 특징

### 1. 📱 스마트폰 OMR 스캔 (NEW!)
- **종이 답안지 자동 인식**: 연필로 작성한 OMR 답안지를 스마트폰으로 촬영하면 자동 인식
- **복수 정답 지원**: 여러 개 칠한 답안도 정확하게 인식
- **하이브리드 입력**: OMR 스캔 + 화면 수정 병행 가능
- **컴퓨터 비전 기술**: Python OpenCV로 정밀한 버블 인식

### 2. 📊 실시간 성적 분석 시스템
- **즉각적인 채점**: 답안 제출 즉시 자동 채점 및 결과 확인
- **단원별 성취도 분석**: 각 단원(물질과 규칙성, 시스템과 상호작용, 변화와 다양성, 환경과 에너지)별 정답률 실시간 제공
- **전체 학생 평균 비교**: 나의 점수와 전체 평균을 비교하여 객관적 위치 파악

### 3. 🎨 화려한 데이터 시각화
- **다중 차트 시스템**: 바 차트, 레이더 차트, 라인 차트를 활용한 입체적 분석
- **색상 코드 피드백**: 성취도에 따른 직관적 색상 시스템 (90점 이상 초록, 70점 이상 파랑, 50점 이상 주황, 50점 미만 빨강)
- **애니메이션 효과**: 성적 표시 시 동적 효과로 성취감 극대화

### 4. 🏆 단원별 비교 우위 분석
학생이 제출한 시험의 각 단원별로:
- **전체 학생 평균점수** 자동 계산
- **전체 최고점수** 실시간 갱신
- **응시 학생 수** 표시로 신뢰도 확인
- 나의 점수와 비교하여 **강점 단원**과 **보완 필요 단원** 한눈에 파악

### 5. 💡 스마트 부분 제출 시스템
- **유연한 시험 응시**: 모든 문제를 풀지 않아도 제출 가능
- **시험범위 외 처리**: 답하지 않은 문제는 "시험범위 외"로 자동 분류 (학생 실수가 아닌 의도적 제외로 간주)
- **복수 정답 지원**: 복수 정답 문제 완벽 대응 (여러 선택지 동시 선택 가능)

### 6. 📈 누적 학습 이력 관리
- **전체 응시 기록**: 학생이 제출한 모든 시험 결과 영구 보관
- **구글 시트 자동 연동**: 모든 시험 결과가 구글 시트에 자동 기록되어 학원과 학생이 언제든 확인 가능
- **성장 추적**: 시간에 따른 성적 변화 추이 분석

## 🚀 시스템 사용 방법

### STEP 1: 로그인
1. 첫 화면에서 **학생 ID**와 **이름**을 입력합니다
2. "로그인" 버튼을 클릭하여 시스템에 접속합니다

**💡 TIP**: 학생 정보는 학원 데이터베이스에 사전 등록되어 있어 안전하게 관리됩니다.

---

### STEP 2: 학교 선택
로그인 후 나타나는 학교 목록에서 **내가 다니는 학교** 또는 **풀고 싶은 학교의 기출문제**를 선택합니다.

**현재 지원 학교**: 16개 학교
- 각 학교별로 실제 기출문제 데이터 완비

---

### STEP 3: 시험 선택
선택한 학교의 기출문제 목록에서 원하는 시험을 고릅니다.

각 시험 카드에는:
- **시험 연도**
- **학기** (1학기 중간고사, 1학기 기말고사 등)
- **과목** (통합과학)

정보가 표시되어 원하는 시험을 쉽게 찾을 수 있습니다.

---

### STEP 4: OMR 답안 작성 (2가지 방법)

#### 📱 방법 1: OMR 답안지 스캔 (NEW!)

시스템에서 제공하는 **OMR 답안지**를 출력하여 연필로 작성한 후 스마트폰으로 촬영하면 자동으로 답안이 인식됩니다!

**스캔 준비:**
1. 시스템에서 **OMR 답안지 템플릿** 다운로드 (A4 용지)
2. 출력 후 연필이나 볼펜으로 동그라미 선택지를 **진하게 칠하기**
3. 복수 정답은 여러 개 칠하면 자동 인식

**스캔 방법:**
1. 시험 페이지 상단의 **📷 카메라로 스캔** 버튼 클릭 (스마트폰의 경우)
2. 또는 **📁 파일 업로드** 버튼으로 촬영한 사진 선택
3. 답안지를 **평평하게** 놓고 **네 모서리**가 모두 보이도록 촬영
4. 자동으로 답안이 인식되어 화면에 체크됩니다!

**💡 스캔 TIP**:
- 조명이 밝은 곳에서 촬영
- 그림자가 생기지 않도록 정면에서 촬영
- 네 모서리 마커(●)가 모두 보이게 촬영
- 스캔 후에도 수동으로 수정 가능

#### 🖱️ 방법 2: 화면에서 직접 선택

**4-1. 진행률 확인**
화면 상단에 **답변한 문제 수 / 전체 문제 수**와 **진행률 바**가 표시됩니다.

**4-2. 답안 선택**
- 각 문제 번호 아래 선택지(①, ②, ③, ④, ⑤)를 클릭하여 답안 선택
- **복수 정답 문제**: 여러 개 선택 가능 (클릭할 때마다 선택/해제 토글)
- 선택한 답안은 **파란색**으로 표시됩니다

**4-3. 부분 제출**
- 모든 문제를 풀지 않아도 **"제출하기"** 버튼 활성화
- 최소 1문제 이상만 답하면 제출 가능
- 답하지 않은 문제는 **시험범위 외**로 자동 처리 (0점 처리되지 않음)

**💡 TIP**: 
- OMR 스캔 + 수동 입력 **병행 가능** (스캔 후 일부 문제만 수정)
- 시간이 부족할 때는 아는 문제만 먼저 제출 가능
- 나중에 다시 응시하여 전체 범위 학습 가능

---

### STEP 5: 결과 확인 - 화려한 분석 화면

제출 즉시 **프리미엄 성적 분석 페이지**로 이동합니다.

#### 📊 메인 대시보드
상단에 큰 성적 카드가 나타나며:
- **성취율 점수** (0-100점)
- **등급 표시** (S, A, B, C, D)
- **성취도에 따른 색상 및 아이콘**

#### 📈 상세 통계 (3개 카드)
1. **정답 수**: 맞힌 문제 개수
2. **응답한 문제 수**: 실제로 답한 문제 수
3. **정답률**: 백분율로 표시

#### 🎯 단원별 성취도 분석 (핵심 기능!)

**탭 1: 바 차트**
- 각 단원별 나의 정답 개수를 막대 그래프로 표시
- 단원별 색상 구분으로 한눈에 비교

**탭 2: 레이더 차트**
- 5개 영역(4개 단원 + 전체)의 균형을 시각화
- 약한 단원이 어디인지 즉시 파악 가능

**탭 3: 상세 분석표**
각 단원별로:
- 📚 **정답/전체 문제 수**
- 📊 **정답률 진행 바**
- 💯 **내 점수**
- 📈 **전체 평균** (이 시험을 푼 모든 학생의 평균)
- 🏆 **최고점** (이 시험을 푼 학생 중 최고 점수)
- 👥 **응시자 수**

**✅ 나보다 높으면**: 초록색 화살표 (↗) - "평균보다 높음"  
**⚠️ 나보다 낮으면**: 빨간색 화살표 (↘) - "평균보다 낮음"

#### 📝 문제별 상세 결과

각 문제마다:
- **문제 번호**
- **내 답안** vs **정답**
- **정/오답 표시** (✓ 초록색 / ✗ 빨간색)
- **복수 정답 여부** 뱃지
- **무응답 문제**: "시험범위 외" 표시

---

### STEP 6: 학습 기록 조회
우측 상단 **"성적 기록"** 버튼을 클릭하면:
- 지금까지 제출한 **모든 시험 결과** 확인
- 각 시험별 **점수, 응답 수, 정답 수, 단원별 분석** 보관
- **시간순 정렬**로 학습 성장 추적

## 🎓 학습 효과

### 1️⃣ 즉각적인 피드백
제출 즉시 결과를 확인하여 **기억이 생생할 때** 오답을 복습할 수 있습니다.

### 2️⃣ 맞춤형 학습 계획
단원별 분석을 통해 **어느 영역을 집중 학습**해야 하는지 명확히 알 수 있습니다.

### 3️⃣ 객관적 위치 파악
전체 평균 및 최고점과 비교하여 **나의 실력을 객관적으로 평가**할 수 있습니다.

### 4️⃣ 동기 부여
화려한 차트와 색상 피드백으로 **학습 성취감**을 극대화하고 **지속적인 학습 동기**를 제공합니다.

### 5️⃣ 효율적 시간 관리
부분 제출 기능으로 **시간이 부족할 때도 아는 문제부터** 풀고 제출할 수 있어 효율적입니다.
