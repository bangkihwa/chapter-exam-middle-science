목동에이원과학학원 학습 평가 시스템
📚 개요

2. 📊 종합 성적 분석

•	단원별 성적표: 각 단원별 상세 성적과 오답 리스트
•	성적 추이 그래프: 시간에 따른 성적 변화를 시각적으로 확인

화려한 데이터 시각화

•	색상 코드 피드백: 성취도에 따른 직관적 색상 시스템 (90점 이상 초록, 70점
 스마트 부분 제출 시스템
•	유연한 시험 응시: 모든 문제를 풀지 않아도 제출 가능
•	시험범위 외 처리: 답하지 않은 문제는 "시험범위 외"로 자동 분류 (학생 실수가 아닌 의도적 제외로 간주)

누적 학습 이력 관리

•	슈퍼베이스 자동 연동: 모든 시험 결과가 구글 시트에 자동 기록되어 학원과 학생이 언제든 확인 가능
•	성장 추적: 시간에 따른 성적 변화 추이 분석
•	4-2. 답안 선택

•	복수 정답 문제: 여러 개 선택 가능 (클릭할 때마다 선택/해제 토글)

•	4-3. 부분 제출
•	모든 문제를 풀지 않아도 "제출하기" 버튼 활성화
•	최소 1문제 이상만 답하면 제출 가능
•	답하지 않은 문제는 시험범위 외로 자동 처리 (0점 처리되지 않음)

📊 메인 대시보드
상단에 큰 성적 카드가 나타나며:
•	성취율 점수 (0-100점)
•	등급 표시 (S, A, B, C, D)
•	성취도에 따른 색상 및 아이콘
📈 상세 통계 (3개 카드)
1.	정답 수: 맞힌 문제 개수
2.	응답한 문제 수: 실제로 답한 문제 수
3.	정답률: 백분율로 표시
🎯 단원별 성취도 분석 (핵심 기능!)
탭 1: 바 차트
•	각 단원별 나의 정답 개수를 막대 그래프로 표시
•	단원별 색상 구분으로 한눈에 비교
탭 2: 레이더 차트
•	5개 영역(4개 단원 + 전체)의 균형을 시각화
•	약한 단원이 어디인지 즉시 파악 가능
탭 3: 상세 분석표 각 단원별로:
•	📚 정답/전체 문제 수
•	📊 정답률 진행 바
•	💯 내 점수
•	📈 전체 평균 (이 시험을 푼 모든 학생의 평균)
•	🏆 최고점 (이 시험을 푼 학생 중 최고 점수)
•	👥 응시자 수
✅ 나보다 높으면: 초록색 화살표 (↗) - "평균보다 높음"
⚠️ 나보다 낮으면: 빨간색 화살표 (↘) - "평균보다 낮음"
📝 문제별 상세 결과
각 문제마다:
•	문제 번호
•	내 답안 vs 정답
•	정/오답 표시 (✓ 초록색 / ✗ 빨간색)
•	복수 정답 여부 뱃지
•	무응답 문제: "시험범위 외" 표시
