# 목동에이원과학학원 OMR 평가 시스템

## 📚 개요

목동에이원과학학원의 **물리학 프리미엄 OMR 평가 시스템**은 학생들의 학습 성취도를 정확하게 측정하고, 즉각적인 피드백을 제공하는 차세대 학습 관리 플랫폼입니다.

---

## ✨ 주요 특징

### 1. 🎯 즉시 채점 시스템
- **실시간 성적 확인**: 시험 제출 즉시 점수와 성취율 확인
- **정확한 오답 분석**: 틀린 문제를 바로 확인하고 복습 가능
- **맞춤형 피드백**: 성취도에 따른 구체적인 학습 조언 제공

### 2. 📊 종합 성적 분석
- **전체 성적 분석**: 모든 단원의 평균 성취율과 학습 추이 확인
- **단원별 성적표**: 각 단원별 상세 성적과 오답 리스트
- **성적 추이 그래프**: 시간에 따른 성적 변화를 시각적으로 확인
- **강점/약점 파악**: 잘하는 단원과 보완이 필요한 단원을 한눈에 파악

### 3. 🔍 편리한 OMR 방식
- **한 페이지에 모든 문제**: 스크롤 없이 전체 문제를 한눈에 확인
- **직관적인 답안 입력**: 클릭 한 번으로 답안 선택 및 수정
- **진행 상황 표시**: 풀지 않은 문제를 실시간으로 확인
- **안전한 제출**: 미완성 답안 경고로 실수 방지

### 4. 📖 16개 물리학 단원 완벽 커버
물리학 프리미엄 교재의 모든 단원을 체계적으로 학습:

**역학 영역**
- 운동의 기술
- 운동 법칙
- 돌림힘 평형과 안정성
- 운동량과 충격량
- 일과 에너지
- 에너지 보존

**열과 파동**
- 열역학
- 파동의 성질
- 파동의 간섭과 회절

**전자기**
- 전기장과 전위
- 물질의 자성 & 전류의 자기작용
- 전자기 유도

**현대물리**
- 에너지 준위와 스펙트럼
- 빛과 물질의 이중성
- 상대성 원리
- 핵반응과 핵에너지

## 👨‍🎓 학생 기능

### 로그인
- 학생 ID와 이름으로 간편하게 로그인
- 개인 정보 보호 및 성적 관리

### 단원 선택 및 시험 응시
- 16개 단원 중 원하는 단원 선택
- 단원별 문제 수 확인 (26~60문제)
- OMR 방식으로 편리한 답안 작성

### 즉시 결과 확인
- 제출 즉시 점수, 성취율, 등급 확인
- 틀린 문제 번호와 정답 확인
- 성취도별 맞춤 피드백

### 성적 리포트
- **전체 성적 분석**
  - 평균 성취율 및 시험 응시 횟수
  - 최고/최저 점수 확인
  - 성적 추이 그래프
  
- **단원별 성적표**
  - 각 단원의 모든 시험 기록 조회
  - 상세 결과 확인 (점수, 등급, 오답)
  - 학습 패턴 분석

## 👨‍💼 관리자 기능

### 학생 관리
- 전체 학생 목록 조회
- 학생별 평균 성적 및 응시 횟수 확인
- 구글 시트 연동으로 학생 정보 자동 동기화

### 학생 성적 조회
- 학생 이름/ID로 빠른 검색
- 개별 학생의 전체 성적 분석
- 단원별 상세 성적 확인
- 학생 리포트와 동일한 화면으로 상담 자료 활용

### 단원별 오답 분석
- **문제별 오답률 분석**
  - 각 문제의 총 응시 인원 및 오답자 수
  - 오답률 높은 순으로 자동 정렬
  - 오답률에 따른 시각적 표시 (빨강/주황/노랑/초록)

- **답안 분포 분석**
  - 1~5번 각 선택지별 선택 인원 통계
  - 정답과 오답 시각적 구분
  - 학생들의 오개념 파악 가능

### 구글 시트 연동
- 학생 정보와 시험 결과 자동 저장
- 문제별 통계 데이터 집계
- 어디서나 구글 시트로 데이터 확인 가능

### 시스템 설정
- 구글 시트 ID 설정 (UI에서 직접 설정)
- 개발/배포 환경 별도 관리
- 설정 상태 실시간 확인
